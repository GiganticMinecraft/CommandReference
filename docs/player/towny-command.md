# Towny Command

プレイヤーが使用できる Towny のコマンドリファレンス。

----

- [Towny Command](#towny-command)

----

## Towny コマンド

### `/towny`

各コマンドのヘルプ一覧を表示

### `/towny ?`

`/towny` 系のコマンドヘルプを表示

### `/towny map`

周辺のマップを表示

### `/towny prices`

町の運営に関わる機能の値段を表示

### `/towny time`

時間を表示。次の税収の時間などの参考になります。

#### `/towny top residents [all|town|nation]`

指定した種類の住民数ランキングを表示

指定しない場合は `all` として実行される

| 引数 | 説明 |
| ---- | ---- |
| `all` | 全ての町と国 |
| `town` | 全ての町 |
| `nation` | 全ての国 |

### `/towny top land [all|town|resident]`

指定した種類の土地数ランキングを表示する

指定しない場合は `all` として実行される

| 引数 | 説明 |
| ---- | ---- |
| `all` | 全ての町と国 |
| `town` | 町ごと |
| `resident` | 住民ごと |
